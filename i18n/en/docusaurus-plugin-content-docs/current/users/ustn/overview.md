# Overview

USTN是TreasureNet的稳定币，波动性最小的代币，创造之初锚定1美元。USTN从一开始就锚定了美元的价值，并保持了相对于美元价值的稳定性，作为一种稳定的货币存在。

USTN的发行方式与现实货币金融体系的调整机制和一系列动态控制方式类似，同时提供存款、贷款等基础金融操作。 这使得USTN能够以与真实货币金融系统相同的方式来保证货币价值的稳定性，从而使其无限接近美元的市场价值。
